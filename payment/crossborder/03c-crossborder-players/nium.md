# Nium Pte. Ltd.（总部新加坡；消费侧汇款品牌 Instarem 为其前身/子品牌）

> 📌 **一句话定位**：面向企业(B2B)的全球实时跨境收付(payout)基础设施提供商，让银行/旅游/薪酬/平台型客户通过一套 API 在全球"收款、换汇、付款、消费"，自述连接 190+ 国、实时到账 100+ 国、持牌 40+ 市场。
> 🏷️ **角色归类**：**跨境付款/清结算分发网络(payout & disbursement) + 嵌入式跨境支付基础设施**——以付款分发为主、兼具全球收款(虚拟账户)能力，**非海外本地收单**（呼应 `03-crossborder-business §7.1`，依附银行体系与卡组织之上的基础设施层，非卡清算组织）。
> ⚠️ **数据时效**：截至 2026-06；规模公司自述、口径前后不一(60B vs 25B)，引用带年份。Wikipedia 条目带利益冲突提示。

## 1. 基本信息
- **成立**：2014 年(起家于消费侧汇款 Instarem，2019 推出/更名 B2B 平台 "Nium")
- **总部**：新加坡(运营重心部分移至旧金山/伦敦)；20+ 办公室
- **创始人**：Prajit Nanu(CEO)、Michael Bermingham(COO)
- **当前状态**：**未上市**(私有)；2023 计划约 2025 中赴美 IPO，2024 下调目标至 2025 甚至 2026 底，IPO 未完成

## 2. 背景与沿革
2014 年以 C 端汇款 Instarem 起家 → 2019 转型 B2B 嵌入式跨境支付平台 Nium → 收购 Ixaris(旅游支付)/Wirecard Forex India/Socash → 2024 down round(估值压缩)、IPO 推迟、2026-02 高管换血。

## 3. 股东与资本 ⚠️down round
- **融资**：Seed(2015)→A(2016,$5M,Vertex)→B(2017,$13M,GSR)→C($38M)→**D(2021,$200M,Riverwood 领投)**→E(2024-06,$50M)
- **投资人**：Global Founders Capital、Vertex、GSR、Fullerton、SBI-FMO、Riverwood、**Visa**、GIC 等
- **估值**：2021 逾 $1B(独角兽)→2022-02 约 $2B→**2024-06 Series E 后约 $1.4B(down round，较前轮跌约 30%)**
- ⚠️ Series E 领投方(Wikipedia 称文莱投资局)未独立核实

## 4. 牌照与资质 ⚠️官网自述、未逐一核实
自述 **40+ 市场持牌**。具体：**日本 FSA"第一类资金移动业"牌照(2024-02，自称首家获此牌照的全球金融科技公司)**；英国 EMI(2020)+发卡(2021)；立陶宛 EMI(2017，欧盟通行)；美国/香港/新加坡/马来西亚/加拿大 MSB 类货币服务牌照；印尼汇款牌照(2019)；新西兰 FSP(2024)。⚠️ "40+ 市场"为公司口径，未逐张核实

## 5. 定位与商业模式 🔧+📌
B2B 嵌入式跨境支付平台。客户(银行/旅游航空/薪酬/平台/AI 场景)经 API 接入，实现全球收款、换汇、向全球派付、发卡消费。盈利：跨境交易/派付手续费、**外汇换汇价差(FX markup)**、发卡及交易分润、平台/账户服务费、合规与多币种账户增值。规模化关键=自持多地牌照+直连各地银行/支付网络压成本、实时到账

## 6. 核心产品与业务范围 📌官网
全球付款/派付(real-time payouts，实时到 100+ 国，连接 190+ 国，覆盖银行/卡/钱包) / 全球收款+多币种虚拟账户 / **发卡平台**(累计发卡 7500 万张/2023，年发卡 Token 41M+) / **Instarem**(C 端跨境汇款子品牌) / **Nium Airline Payments**(源 2021 收购 Ixaris，自称全球第三大 B2B 旅游卡发行方) / Crypto/稳定币基础设施(2021 推 Crypto-as-a-Service)

## 7. 业务区域
自述连接 190+ 国、实时派付 100+ 国；20+ 办公室(旧金山/伦敦/阿姆斯特丹/新加坡/东京/印度/澳洲/香港/马来西亚等)；主服务亚太、北美、欧洲
