# PingPong（运营主体「杭州呯嘭智能技术有限公司」）

> 📌 **一句话定位**：起家于"帮中国跨境电商卖家把海外平台的钱收回国"的第三方跨境收款服务商，现扩展为覆盖收款(Pay In)+付款(Pay Out)+外汇+收单+发卡的一体化跨境支付基础设施提供商。
> 🏷️ **角色归类**：**以"跨境收款"起家，现"两者都做"，但收款仍是核心主业**（呼应 `03-crossborder-business §7.1`）。官网已列 Acquiring(收单)+Card Issuing(发卡)，宣传"全球收单"，但收单实际规模/持牌深度未独立核实，可能更多是聚合/合作而非全部自持本地收单牌照。
> ⚠️ **数据时效**：截至 2026-03；规模数字均公司自述口径、不同年份差异大，引用务必带年份。

## 1. 基本信息
- **成立**：2015 年 📌（多源一致；中文源细到 2015-05/06，2014-08 起筹建）
- **总部**：中国**杭州**(滨江) 📌（英文数据库把"PingPong Global Solution Inc."记为纽约，系美国运营实体；集团与研发总部在杭州）
- **创始人**：**陈宇（Yu Chen）** 📌（1980 年生于湖北随州，复旦本科+美国康涅狄格大学数学硕士）
- **当前状态**：**未上市**；2020 年曾启动深交所创业板 IPO 辅导(中信证券)，至今无上市消息 📌

## 2. 背景与沿革
2015 年创立于杭州，以"亚马逊卖家收款"切入跨境收款，迅速成为中国跨境电商收款头部。逐步扩展付款/外汇/收单/发卡，2019 获欧洲支付牌照，经收购信航支付拿中国本土牌照（2026-03 实现 100% 全资控股），2025 拓中东(UAE)+东南亚(马来西亚)。

## 3. 股东与资本 ⚠️多源不一致

- **融资**（中文源，轮次计数 6~9 轮不一）：天使(2015, 金石投资/中信证券投资) → B/B1(2017, 广发信德领投, 富达亚洲参投, 数亿元) → C(2018, 约 7 亿元, 华睿投资等) → 股权融资(2021, 广发信德等) → D(投资方含华睿/鸥翎/赛伯乐/宽带资本CBC/普罗资本)；早期亦有 IDG

## 5. 定位与商业模式
- **产业链位置**：跨境支付的"卖家侧收款+跨境资金通道"中间层，夹在海外电商平台/海外收单行与中国卖家之间
- **怎么赚钱**：① 收款提现费率(从亚马逊等回款按比例收，行业典型 0.x%~1% 级，未逐项核实) ② 货币兑换汇差(30 种货币、400+ 货币对) ③ B2B 外贸付款通道费 ④ 增值(发卡/供应链金融/VAT 税务/SaaS/全球分销)

## 6. 核心产品与业务范围
跨境收款(Pay In, 起家主业, 服务亚马逊/Shopify/TikTok Shop/Temu/SHEIN 卖家) / 全球付款(Pay Out) / 外汇(30 种持有货币/23 种本地账户币种/400+货币对) / 收单解决方案(Acquiring) / 发卡(Card Issuing) / 多币种本地收款账户 / 供应链金融+VAT 税务+SaaS / 可定制 API。2025 与 Visa 合作 card-to-account 企业付款(BPSP, 170 国/25 币种)
